## REQUIREMENTS

- Git.
- Node 8.9 or Higher.
- Angular CLI 6.1.4 or Higher.
- NPM 5.5.1 or higher.


## WALKTHROUGH

1) Clone repository.
2) Open command line and navigate to cloned repository.
3) Run npm install.

  ## DEPLOY IN SERVER (AFTER WALKTHROUGH)
1) Run ng build --prod
2) Copy files in dist folder to the desired server.

  ## DEPLOY LOCALLY (AFTER WALKTHROUGH)
1) RUN ng serve -o.
